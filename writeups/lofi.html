// Overview

Lo-Fi is a TryHackMe room that shows a classic Local File Inclusion (LFI) vulnerability. The goal is simple: go to the provided URL and find the flag in the root of the filesystem.

Though the room is rated "Easy," it teaches an important lesson about improper input sanitization. This bug still affects web applications today.

// The Vulnerability: LFI via ?page=

Visiting the provided URL shows a straightforward music-themed page. Looking at the page source uncovers an important clue in the sidebar: 

<!-- Sidebar Widgets Column -->
<div class="card my-4">
  <h5 class="card-header">Discography</h5>
  <div class="card-body">
    <ul class="list-unstyled mb-0">
      <li><a href="/?page=relax.php">Relax</a></li>
      <li><a href="/?page=sleep.php">Sleep</a></li>
      <li><a href="/?page=chill.php">Chill</a></li>    
      <li><a href="/?page=coffee.php">Coffee</a></li>
      <li><a href="/?page=vibe.php">Vibe</a></li>
      <li><a href="/?page=game.php">Game</a></li>
    </ul>
  </div>
</div>

This ?page= parameter is a classic sign of Local File Inclusion (LFI). The application probably uses PHP's include() function without proper sanitization.

Testing the parameter with ../../../../etc/passwd confirms the vulnerability:

Interestingly, trying to read /flag.txt directly does not work, which indicates a basic filter for absolute paths. However, using relative traversal with ../ completely bypasses this check.

// Key Takeaways

Always check the page source first. The vulnerability was visible in plain HTML; no directory enumeration was needed. This is a reminder to begin with the simplest reconnaissance.
Absolute path filters are easy to bypass. The application checked for a slash at the start, but ../ sequences got around it completely. This is a common mistake in input sanitization.
LFI is more than just a file reader. While this room only required reading a file, LFI vulnerabilities can often be combined with other techniques, like log poisoning, to enable remote code execution. This makes them an important finding in any assessment.
